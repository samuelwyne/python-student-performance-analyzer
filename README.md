# Python Student Performance Analyzer

## Description

This project analyzes student performance using Python lambda functions. It sorts students by marks, filters passing students, and calculates useful class statistics.

## Features

- Store student names and marks
- Rank students from highest to lowest
- Find the top-performing student
- Find the lowest-performing student
- Calculate the class average
- Filter passing students

## Technologies Used

- Python 3

## Concepts Practiced

- Lambda Functions
- map()
- filter()
- sorted()
- Lists
- Dictionaries

## Future Improvements

- Export results to CSV
- Save rankings to JSON
- Generate performance reports
- Add charts for visualization
